## Blue Team Incident Response Lab

This project simulates a real-world cybersecurity incident and demonstrates how a Blue Team responds, investigates, and mitigates the threat.

---

##  Project Overview

The lab follows a phishing attack that leads to suspicious command execution on a user endpoint.

The project walks through:
- Detection  
- Triage  
- Investigation  
- Containment  
- Lessons learned  

---

##  Objectives

- Simulate incident response workflow  
- Practice triage and investigation  
- Document response actions  
- Improve detection and prevention  

---

##  Scenario

A user receives a phishing email, opens an attachment, and triggers suspicious PowerShell activity.

---

##  SOC / IR Relevance

This project reflects real-world tasks:
- Alert triage  
- Incident investigation  
- Response actions  
- Reporting  

---

##  Structure

- incident → full report  
- timeline → attack timeline  
- triage → initial analysis  
- response → containment  
- lessons_learned → improvements  
- playbooks → reusable procedures  

---

## 📌 Key Insight

Incident response is about speed, accuracy, and structured thinking.
