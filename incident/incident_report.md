# Incident Report

## Summary

A phishing attack led to suspicious PowerShell execution on a user endpoint.

---

## Impact

- One endpoint affected  
- Potential credential exposure  

---

## Root Cause

User executed a malicious attachment from a phishing email.

---

## Detection

- Alert triggered on suspicious PowerShell execution  

---

## Response

- Endpoint isolated  
- Investigation conducted  
- No further spread detected  

---

## Conclusion

The incident was contained quickly, but highlights the need for better user awareness and detection.
